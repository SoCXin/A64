## [芯片描述](https://github.com/sochub/A64/wiki) 

珠海全志 [allwinner](https://github.com/sochub/allwinner) cortex A53四核64位应用处理器，最早将价格推进到5美元以下的4核64位CPU，产品本身无太多亮点，芯片的外设配置极简

主要特点：性价比和存量市场

器件封装：BGA396 15mm x 15mm （0.65mm pitch）
